# Code of Conduct

This project uses a lightweight contributor covenant: be respectful, constructive, and focused on making the software safer and more useful.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different experience levels and viewpoints.
- Give technical feedback that is specific, actionable, and relevant.
- Assume good intent, but accept correction when behavior has a negative impact.

## Unacceptable Behavior

- Harassment, discrimination, threats, or personal attacks.
- Sexualized language or imagery in project spaces.
- Publishing private information without explicit permission.
- Repeated disruption of issues, pull requests, or discussions.

## Enforcement

Maintainers may remove comments, close discussions, block participants, or take other reasonable action to protect the project community. Report concerns privately to a maintainer when possible.
